소놀비서 개인정보처리방침

시행일: 2026년 2월 20일

1. 개요

소놀비서(이하 "앱")는 시니어 친화적 AI 음성 비서 앱입니다. 본 개인정보처리방침은 앱이 수집하는 정보와 그 사용 방법을 설명합니다.

2. 수집하는 정보

2.1 음성 데이터 (마이크 권한)

  • 앱은 음성 인식(STT) 기능을 위해 마이크 권한을 요청합니다.
  • 음성 데이터는 Google 음성 인식 서비스를 통해 텍스트로 변환됩니다.
  • 음성 데이터는 기기에서 실시간 처리되며, 앱 서버에 저장되지 않습니다.

2.2 대화 데이터

  • AI 응답을 위해 사용자의 텍스트 입력이 Google Gemini API로 전송됩니다.
  • 대화 내역은 기기 내에서만 유지되며, 외부 서버에 영구 저장되지 않습니다.

2.3 위치 정보

  • 날씨 조회 기능 사용 시 대략적인 위치 정보가 활용될 수 있습니다.
  • 위치 정보는 날씨 API 호출에만 사용되며 저장되지 않습니다.

3. 정보의 사용 목적

  • 음성 명령을 텍스트로 변환 (음성 인식)
  • AI 기반 대화 응답 생성
  • 날씨 정보 제공

4. 제3자 서비스

  • Google Gemini API: AI 대화 처리
  • Google 음성 인식: 음성-텍스트 변환
  • Open-Meteo API: 날씨 정보 제공

각 서비스의 개인정보처리방침은 해당 서비스 제공자의 정책을 따릅니다.

5. 데이터 보안

앱은 사용자 데이터를 자체 서버에 수집하거나 저장하지 않습니다. 모든 데이터는 기기 내에서 처리되거나, 위 제3자 서비스를 통해 실시간으로만 처리됩니다.

6. 아동 개인정보

앱은 만 14세 미만 아동의 개인정보를 의도적으로 수집하지 않습니다.

7. 개인정보처리방침 변경

본 방침은 필요에 따라 변경될 수 있으며, 변경 시 앱 내 또는 이 페이지를 통해 공지합니다.
